# Star Parkings

Affiche en live les places disponibles dans les parkings Star

## Guide du développeur

### Dépendances 

- Python 3.10

```shell

git clone https://github.com/VirgileDelcourt/StarParkings.git
cd StarParkings
pip install --requirement requirements.txt
```

### Lancer le serveur

```shell
python main.py
```
